# Taxi du Môle - Site Next.js

Site professionnel pour Taxi du Môle, construit avec Next.js, Tailwind CSS et Framer Motion.

## Pages principales

- Accueil
- Services
- Transferts aéroports et gares
- Zones desservies Haute-Savoie
- Points forts
- Démarches de réservation
- Solutions
- Preuves
- FAQ
- Ressources
- Contact
- Mentions légales
- Politique de confidentialité
- Sitemap XML
- Robots.txt

## Commandes

```bash
npm run dev
npm run build
npm start
```

## Déploiement

Le site est prêt pour GitHub et Vercel.
